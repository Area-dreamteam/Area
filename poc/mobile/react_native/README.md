# React-Native POC
## Only front-end with a Login and Register Page

### Objective

- A simple login and register form
- No back-end connection and no database, API
- Basic page redirection with the path of the page when the user submit
- And a final page with Connect or Registered

### Installation and Run

To run the project, navigate to the directory and run one of the following npm commands:

``` bash

- cd poc

- npm run android
# or
- npm run ios # you need to use macOS to build the iOS project - use the Expo app if you need to do iOS development without a Mac
# or
- npm run web
# or
- npx expo start

```
